
# Inked

shadcn color themes registry

## Usage

Add the registry to your `components.json`

```json
"registries": {
  "@inked": "https://aitorllamas.com/inked/public/r/{name}.json"
}
```

Then you can install any of the themes

```sh
pnpm dlx shadcn@latest add @inked/flexoki
```
